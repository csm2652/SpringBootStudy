# SpringBootStudy

첫 웹개발 시작으로

springboot + gradle + mysql을 이용하여 간단하게 게시판 만들기를 따라해봤다.

> 출처: https://kyuhyuk.kr/category/spring-boot.html

### 개발 도구

+ IDE: inteliJ

* Framework: SpringBoot
* Database: Mysql
* Build tool: Gradle

***



### 주요 기능

+ 게시글 등록
  + 제목
  + 작성자
  + 게시글
  + 첨부파일 등록
+ 게시글 삭제
+ 게시글 수정
